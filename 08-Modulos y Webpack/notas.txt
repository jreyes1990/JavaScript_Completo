*****   Enlace de webpack
        https://webpack.js.org/

        https://v4.webpack.js.org/loaders/


*****   Instalar lo siguiente, si en caso aun no esta instalado
        *****   Verificar version de node js
                node -v

        *****   Verificar version de npm
                npm -v

        *****   Creacion de archivo package.json
                npm init

*****   Instalacion de webpack
        *****   Linea de comando
                npm install webpack webpack-cli --save-dev

*****   Aplicar configuraciones de webpack
        npm run build    

*****   Crear archivo de configuracion webpack
        webpack.config.js

*****   Instalacion de Dev-Server, sus configuraciones en package.json
        npm i -D webpack-dev-server

*****   Ejecutando el server
        npm run start

*****   Instalacion de css webpack
        npm install --save-dev style-loader

        npm install --save-dev css-loader

        npm install --save-dev css-loader style-loader

*****   Configuracion para crear archivos de forma global
        npm install --save-dev mini-css-extract-plugin

*****   Configuracion para la carga de imagenes
        npm install file-loader --save-dev

        npm install copy-webpack-plugin --save-dev
